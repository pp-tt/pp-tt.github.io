Java 集合

Posted on 2022-04-07

集合

集合概念

对象的容器，实现了对对象的常用操作

集合和数组的区别

  1. 数组长度固定，集合长度不固定
  2. 数组可以存储基本类型和引用类型，集合只能存储引用类型

Collection

特点

代表一组任意类型的对象，无序、无下标、不能重复。

方法

  • boolean add(Object obj)
  • boolean addAll(Collection c)
  • void clear()
  • boolean contains(Object obj)
  • boolean equals(Object obj)
  • boolean isEnpty()
  • boolean remove(Object obj)
  • boolean size()
  • Object[] toArray()
